한국 D2C 역직구 체크아웃, 18줄.

주소·배송·결제·약관 4개 위젯을 한 페이지에서 직접 시연합니다. 좌측에서 옵션을 바꾸면 우측 4개의 마운트가 즉시 반응하고, 위젯이 발행하는 모든 이벤트는 좌측 로그에 그대로 쌓입니다. 키 없이 동작하며, 표시된 코드를 그대로 복사해 자기 사이트에 붙이면 됩니다.

SDK 번들을 불러오지 못했습니다. pnpm docs:build로 docs/pages/assets/opencheckout-widget.js를 생성했는지 확인하세요.
